# Recast

Recast is a high-performance open-source screen recording tool with integrated editing capabilities. 

## Features
- Screen capture
- Audio recording
- Timeline-based video editor
- Video exports

## Developing

Once you've created a project and installed dependencies with `npm install` (or `pnpm install`, `yarn`, `bun`), start a development server for the UI:

```sh
npm run dev
```

For the desktop app (Tauri):
```sh
npm run tauri dev
```

## Building

To create a production version of your app:

```sh
npm run build
npm run tauri build
```
